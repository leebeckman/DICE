2.1.8

<p>Over 120 features have been added, changed or fixed in JForum 2.1.8, like:</p>

<ul>
<li>Lucene as search engine</li>
<li>Replaced DWR with JQuery for all AJAX calls</li>
<li>Inline message editing</li>
<li>New BB Tags</li>
<li>Administrators can now manually approve an user account</li>
<li>Identification of Moved Topics</li>
<li>"Search this forum" search</li>
<li>Record of how many times a message was edited</li>
<li>Logging of Moderation Activity</li>
<li>RSS now shows a snippet of the posted message</li>
</ul>

<p>
Many important bugs where fixed. A more comprehensive description of what's new in JForum 2.1.8 is available in the page
<a href="http://www.jforum.net/doc/NewFeatures2.1.8"> New and changed features in JForum 2.1.8</a>.
</p>
